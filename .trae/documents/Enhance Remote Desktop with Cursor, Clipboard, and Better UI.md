我将升级远程桌面工具，满足您提出的四个核心需求：可见鼠标、文字输入、剪贴板同步以及高用户体验。

## 1. 鼠标位置可见 (服务器端优化)
- **目标**：确保在远程画面中能清晰看到鼠标光标的位置。
- **实现**：
    - 在 `server.py` 中引入图像绘图库 (`PIL.ImageDraw`)。
    - 每次截屏后，使用 `pyautogui.position()` 获取当前鼠标坐标。
    - 在发送图像前，人工在图像上绘制一个醒目的光标（如红色圆点），确保画面与鼠标位置完美同步。

## 2. 剪贴板复制粘贴 (双向同步)
- **目标**：支持在本地电脑和远程电脑之间复制粘贴文本字符。
- **实现**：
    - **发送到远程**：在客户端增加“发送剪贴板”功能。将本地剪贴板内容发送给服务器，服务器通过 `pyperclip.copy()` 写入远程电脑剪贴板。
    - **从远程获取**：在客户端增加“获取剪贴板”功能。服务器读取远程剪贴板内容发送回浏览器，用户点击确认后写入本地剪贴板。

## 3. 远程文字输入 (可靠输入)
- **目标**：解决特殊字符输入困难的问题，支持长文本输入。
- **实现**：
    - 在客户端工具栏增加一个“远程输入框”。
    - 用户可以在框内输入或粘贴一段文字，点击发送后，服务器端使用 `pyautogui.write()` 自动将这段文字“敲”进去。这比单纯的按键映射更可靠，特别适合输入密码或网址。

## 4. 提升用户体验 (全新 UI)
- **目标**：操作更便捷，界面更友好。
- **实现**：
    - **悬浮工具栏**：设计一个半透明的悬浮菜单，集成所有功能（剪贴板、输入框、设置）。
    - **状态指示**：添加连接状态灯（在线/离线）。
    - **快捷键优化**：优化 Command/Ctrl 等修饰键的映射，让远程操作更顺手。

2.  **重写 `static/index.html`**：实现全新的悬浮 UI 界面，对接新的 WebSocket 指令。
3.  **验证**：测试鼠标跟随是否流畅，剪贴板是否互通，文字输入是否准确。
