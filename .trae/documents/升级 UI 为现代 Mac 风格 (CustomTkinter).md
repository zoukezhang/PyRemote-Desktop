为了打造**高端、现代、Mac 风格**的 UI，我将引入 `customtkinter` 库。这是一个专为 Python 设计的现代 UI 库，支持圆角、透明度、深色模式和高 DPI 渲染，能完美复刻 macOS 原生应用的质感。

## 核心设计理念 (Modern Mac-Style)
1.  **视觉风格**：
    *   **圆角卡片式布局**：将信息（IP、验证码）和设置（画质、帧率）分块放入悬浮感的圆角卡片中。
    *   **自适应深色模式**：自动跟随系统切换黑/白主题，呈现高级的磨砂质感。
    *   **现代控件**：使用**开关 (Switch)** 代替老式复选框，使用**圆角按钮**代替矩形按钮。
2.  **布局重构**：
    *   **顶部 (Hero Area)**：超大状态指示灯 + 醒目的“启动/停止”开关。
    *   **中部 (Connection)**：醒目的连接地址和验证码，配备“一键复制”图标。
    *   **底部 (Settings)**：折叠式或标签式的设置面板，保持界面整洁。

## 执行步骤
1.  **安装依赖**：
    *   需要运行 `pip install customtkinter`（我会尝试在代码中自动检测并安装，或者您手动运行）。
2.  **重写 `server.py`**：
    *   引入 `customtkinter`。
    *   移除所有老旧的 `tk` 和 `ttk` 控件，替换为 `CTkButton`, `CTkFrame`, `CTkEntry` 等。
    *   重新设计布局代码，增加 `pady`/`padx` 留白，提升呼吸感。
3.  **重写 `client.py` (可选但推荐)**：
    *   让客户端也拥有一致的现代风格。

我们将先从 **服务端 (`server.py`)** 开始改造。
